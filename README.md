# zip-image-viewer

An Electron application using React and TypeScript. This application is used to view images from a zip file without extracting them.

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```


## requirement
- Electron > 28.3.3
- node > 18.18.2
- React > 18.3.1
